# DebugPrint

C++20 ヘッダーオンリーのデバッグ出力ライブラリ。

## デモ

WebAssembly版をブラウザで試せます：
**[オンラインデモ](https://yummy5678.github.io/MyDebugFunctions/)**

## 機能

- メッセージ出力（通常・警告・エラー）
- 変数・構造体の自動表示（Boost.PFR / magic_enum）
- 関数・クラスのトレース
- ポップアップダイアログ（ネイティブ: tinyfiledialogs / Wasm: SweetAlert2）
- カラー出力対応
- ログファイル書き出し
- 多言語対応（日本語・英語）
